# AI 聊天网站 UI/UX 优化需求

## 项目背景
当前 AI 聊天网站采用左右分栏布局（深色侧边栏 + 浅色主内容区），基础功能已具备，但视觉设计、交互体验和细节处理需要全面优化。

---

## 一、侧边栏区域（左侧深色面板）

### 问题点 1.1：顶部品牌区
**当前状态：** "AI助手" 文字 + 汉堡菜单图标，视觉平淡
**具体位置：** 侧边栏最顶部，固定高度区域

**改进项：**
- 将 "AI助手" 升级为 Logo 图形 + 文字组合
- 增加品牌色点缀（建议使用紫色/蓝色渐变）
- 汉堡菜单改为更现代的 **设置齿轮图标** 或 **用户头像**
- 添加悬停提示 "展开/收起侧边栏"

### 问题点 1.2：新建对话按钮
**当前状态：** 灰色背景按钮，"+ 新对话" 文字，视觉权重不足
**具体位置：** 品牌区下方，搜索框上方

**改进项：**
- 改为**主色调填充按钮**（紫色/蓝色）
- 图标使用 **加号/铅笔组合图标**，增加创意感
- 添加 **快捷键提示**（如 "Ctrl+N"）在悬停时显示
- 增加**微动效**：悬停时轻微上浮 + 阴影加深

### 问题点 1.3：搜索框
**当前状态：** 纯灰色背景，放大镜图标，无交互反馈
**具体位置：** 新建对话按钮下方

**改进项：**
- 改为**圆角搜索栏**，白色/浅灰背景
- 占位符文字："搜索对话历史..."
- 聚焦时**外发光边框**（品牌色）
- 增加**清除按钮**（输入内容后显示）
- 支持 **Cmd/Ctrl + K** 快捷键聚焦

### 问题点 1.4：对话列表项
**当前状态：** 灰色背景条目，仅显示标题和时间，选中态不明显
**具体位置：** 搜索框下方的可滚动区域

**改进项：**
- 选中项：左侧增加 **3px 品牌色竖条指示器**
- 选中项背景：从灰色改为 **品牌色 10% 透明度**
- 悬停态：显示**操作按钮组**（置顶、重命名、删除）
- 对话标题：支持**双击编辑**，回车保存
- 时间格式：相对时间（"5分钟前" → "刚刚"、"昨天"）
- 增加**对话图标**：根据内容自动显示（代码/图片/文档）

### 问题点 1.5：底部功能区
**当前状态：** 截图未显示，但通常缺少用户相关入口
**具体位置：** 侧边栏最底部

**改进项：**
- 添加**用户头像 + 用户名**区域
- 展开菜单包含：个人设置、主题切换、帮助中心、登出

## 二、顶部导航栏（主内容区顶部）

### 问题点 2.1：标题区域
**当前状态：** 仅显示 "新对话"，左侧空白
**具体位置：** 主内容区最顶部，左侧

**改进项：**
- 当前对话标题改为**可编辑输入框**样式
- 默认显示 "新对话"，点击即可重命名
- 增加**对话状态标签**（如 "已保存"、"草稿"）

### 问题点 2.2：顶部工具栏
**当前状态：** 右侧仅有主题切换、历史、分享图标，分散排列
**具体位置：** 主内容区顶部右侧

**改进项：**
- 整合为**统一的工具栏组件**：
  - 模型选择下拉框（显示 "Qwen 3.5 Plus" + 下拉箭头）
  - 分隔线
  - 分享按钮（带下拉菜单：复制链接、导出PDF、邮件发送）
  - 主题切换按钮（太阳/月亮图标切换）
  - 更多操作（三点菜单：清空对话、删除对话、设置）

### 问题点 2.3：缺少面包屑/导航
**当前状态：** 无返回路径，用户易迷失
**具体位置：** 标题区域左侧

**改进项：**
- 当进入特定功能（如设置页）时显示面包屑
- 格式：首页 &gt; 设置 &gt; 个人资料

---

## 三、消息展示区（主内容区中部）

### 问题点 3.1：AI 回复头部
**当前状态：** "思考过程" 折叠项样式简单，箭头图标基础
**具体位置：** 每条 AI 消息顶部

**改进项：**
- 设计**精美的思考过程容器**：
  - 背景：浅灰色圆角卡片
  - 图标：灯泡/大脑图标 + "思考过程" 文字
  - 展开动画：平滑高度过渡 + 内容淡入
  - 收起时显示摘要："已思考 3 个步骤"
- 思考中状态：**脉冲动画 + "正在思考..." 文字**

### 问题点 3.2：数学公式展示
**当前状态：** 公式显示为纯文本 "L = Lcls + 0.5 * (LCLIP + Lcls')"，可读性差
**具体位置：** AI 回复内容中的公式段落

**改进项：**
- 使用 **KaTeX 或 MathJax** 渲染数学公式
- 行内公式：$...$ 包裹，使用专业数学字体
- 块级公式：$$...$$ 包裹，居中显示，带编号
- 支持公式**点击放大**查看
- 深色模式下自动切换为浅色公式

### 问题点 3.3：文本排版
**当前状态：** 正文文字密集，层级不清
**具体位置：** AI 回复的主要内容区域

**改进项：**
- 标题层级：**加粗 + 字号区分**（问题点/改进项作为 H3）
- 列表项：使用**圆点/数字列表**，增加缩进
- 关键术语：**高亮背景色**或**彩色文字**（如 "总损失" 用品牌色）
- 行高：从 1.5 增加到 **1.7**
- 段落间距：增加到 **1em**

### 问题点 3.4：代码块展示
**当前状态：** 无代码块样式（当前截图无代码，但需预留）
**具体位置：** AI 回复中的代码片段

**改进项：**
- 代码块容器：
  - 顶部栏：语言标签（如 "python"）+ 复制按钮 + 下载按钮
  - 背景：深色主题（VS Code 风格）
  - 字体：等宽字体（Fira Code / JetBrains Mono）
  - 行号：可选显示
- 行内代码：浅灰背景 + 圆角 + 等宽字体

### 问题点 3.5：消息气泡样式
**当前状态：** 用户和 AI 消息视觉区分度不足
**具体位置：** 每条消息的容器

**改进项：**
- 用户消息：
  - 背景：品牌色渐变（紫色 → 蓝色）
  - 文字：白色
  - 圆角：右上圆角较小（指向用户）
  - 最大宽度：70%，右对齐
- AI 消息：
  - 背景：透明/白色
  - 左侧增加 **AI 头像/Logo**
  - 圆角：左上圆角较小（指向 AI）
  - 最大宽度：85%，左对齐

## 四、输入区域（主内容区底部）

### 问题点 4.1：输入框容器
**当前状态：** 底部输入区域过于简单，功能入口不明显
**具体位置：** 主内容区最底部，固定高度

**改进项：**
- 改为**悬浮卡片式设计**：
  - 背景：白色/深色卡片
  - 阴影：上方投影，与内容区分离
  - 圆角：16px 大圆角
  - 边框：1px 浅灰边框，聚焦时变为品牌色

### 问题点 4.2：多模态输入入口
**当前状态：** 仅有基础图标，排列松散
**具体位置：** 输入框左侧/下方

**改进项：**
- 左侧工具栏（图标按钮组）：
  - 图片上传（相册图标，支持拖拽）
  - 文件上传（回形针图标，支持多文件）
  - 语音输入（麦克风图标，录音动画）
  - 联网搜索（地球图标，开关状态）
  - 快捷指令（"/" 图标，点击展开菜单）
- 每个图标悬停显示**功能提示 + 快捷键**

### 问题点 4.3：输入框本体
**当前状态：** 单行输入，不支持多行文本
**具体位置：** 输入区域中央

**改进项：**
- 支持**多行文本输入**，自动增高（最小 1 行，最大 5 行）
- 占位符文字："输入消息，Shift+Enter 换行..."
- 支持 **Markdown 快捷输入**（如 # 自动转为标题样式预览）
- 输入时显示**字符计数**（如 "0/2000"）

### 问题点 4.4：发送按钮
**当前状态：** 蓝色圆形按钮，固定位置
**具体位置：** 输入框右侧

**改进项：**
- 空输入状态：灰色禁用态，降低透明度
- 有输入后：品牌色激活态，**轻微脉动动画**吸引点击
- 发送中状态：改为 **加载动画**（旋转圆环）
- 支持 **Enter 发送 / Shift+Enter 换行**（可切换）

## 五、全局与细节优化

### 问题点 5.1：空状态设计
**当前状态：** 新对话时显示空白或简单文字
**具体位置：** 无消息时的主内容区

**改进项：**
- 中央展示**欢迎卡片**：
  - 大标题："今天想聊点什么？"
  - 副标题："我可以帮你写作、编程、分析数据、解答问题..."
  - 快捷模板网格（3-4 个卡片）：
    - "📝 帮我写一封邮件"
    - "💻 解释这段代码"
    - "📊 分析这份数据"
    - "🎨 设计一个 logo 方案"

### 问题点 5.2：滚动与加载
**当前状态：** 长对话滚动体验生硬
**具体位置：** 消息列表容器

**改进项：**
- 平滑滚动（CSS scroll-behavior: smooth）
- 滚动到顶部时**自动加载历史消息**（分页加载）
- 新消息到达时，智能判断是否需要**自动滚动到底部**
- 增加**滚动到底部按钮**（当用户向上滚动查看历史时显示）

### 问题点 5.3：响应式适配
**当前状态：** 未考虑移动端
**改进项：**
- 移动端侧边栏改为**抽屉式导航**（从左侧滑出）
- 输入区域固定在底部，**避开输入法弹出**
- 消息气泡宽度调整为 **90%**

### 问题点 5.4：深色模式
**当前状态：** 仅有简单的主题切换
**改进项：**
- 完整的深色配色方案：
  - 背景：#1a1a2e（深蓝黑）
  - 卡片：#16213e（深蓝灰）
  - 文字：#e94560（强调色）+ #f5f5f5（主文字）
- 自动跟随系统主题设置
- 切换动画：平滑颜色过渡（0.3s ease）

## 六、交互动效规范

| 场景 | 动效描述 | 时长 | 缓动函数 |
|------|---------|------|---------|
| 侧边栏展开/收起 | 宽度变化 + 内容淡入淡出 | 300ms | ease-in-out |
| 消息发送 | 气泡从底部滑入 + 渐显 | 200ms | ease-out |
| AI 流式输出 | 文字逐字出现，光标闪烁 | 20ms/字 | linear |
| 按钮悬停 | 上浮 2px + 阴影加深 | 150ms | ease |
| 思考过程展开 | 高度展开 + 内容淡入 | 250ms | ease-out |
| 主题切换 | 颜色渐变过渡 | 300ms | ease-in-out |
| 加载状态 | 旋转动画 + 脉冲效果 | 1s/圈 | linear |
